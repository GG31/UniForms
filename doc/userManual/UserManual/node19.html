Zone de ``Paramètres''

Trois champs sont paramètrables, figure  :
Imprimable
: à cocher si vous souhaitez imprimer le formulaire
Anonyme
: à cocher si vous ne souhaitez pas définir de destinataire en particulier. Ce formulaire sera donc en accès libre à toutes les personnes disposants de son lien url.
Nombre de réponses max
: Vous pouvez définir le nombre de réponse possible pour le formulaire. Si vous mettez ``3'', chaque destinataire pourra répondre trois fois à ce même formulaire. Si vous souhaitez que le nombre de réponses soit infini, mettez ``0''.
